Markdown

Markdown Wiki

Markdown is a markup language with plain text formatting syntax designed so that it can be converted to HTML and many other formats using a tool by the same name. Markdown is often used to format readme files, for writing messages in online discussion forums, and to create rich text using a plain text editor.

Vim

Vim-instant-markdown(option)

Want to instantly preview finnicky markdown files, but don’t want to leave your favorite editor, or have to do it in some crappy browser textarea? vim-instant-markdown is your friend! When you open a markdown file in vim, a browser window will open which shows the compiled markdown in real-time, and closes once you close the file in vim.

Install vim-instant-markdown

vim-instant-markdown

The following steps are used on Mac OS X.

  1. Install node.js
    brew cask install node
  2. Install instant-markdown-d
    sudo npm -g install instant-markdown-d
  3. Install vim-instant-markdown plugin
    Copy the after/ftplugin/markdown/instant-markdown.vim file from this repo into your ~/.vim/after/ftplugin/markdown/
  4. Ensure you have the line filetype plugin on in your .vimrc
  5. Add let g:instant_markdown_slow = 1 in your .vimrc
  6. Make .md as markdown file (add in your .vimrc) autocmd BufNewFile,BufReadPost .md set filetype=markdown
